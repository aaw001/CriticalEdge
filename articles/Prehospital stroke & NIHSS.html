Prehospital Stroke & NIHSS

Why It Matters

Every minute counts in stroke. Neurons die rapidly, and delays translate into permanent disability. The prehospital team’s role is recognition, rapid triage, and preparation for reperfusion therapy.

Time Goal: Transport should begin within 25 minutes of first medical contact.

Primary Actions

  • Primary and secondary survey
  • Identify symptom onset or last-known-normal
  • Vital signs: SpO₂, RR, HR, BP, glucose, temperature
  • O₂ if SpO₂ <90% (target 94–96%, or 88–92% if COPD)
  • Check for anticoagulation and bleeding risks
  • Evaluate with NIHSS/eStroke
  • Connect multi-monitor and begin live transmission
FAST-VAN exam → Suspected stroke?
Establish onset time (or wake-up stroke)
Check contraindications (bleed risk, anticoagulation)
Decide pathway: Thrombolysis <4.5h | EVT up to 24h
Transport to nearest stroke/EVT center

Supportive Measures

  • Elevate head 30°
  • Insert 2 IV lines if possible (18G preferred)
  • Slow infusion: Ringer’s acetate or NaCl 0.9%
  • Treat fever (>37.5 °C → Paracetamol)
  • Treat hyperglycemia >10 mmol/L
  • Control SBP if >220 mmHg
  • Stop seizures: Midazolam 5 mg
Remember: Stroke ≠ only clot. About 15% are hemorrhages—CT imaging is required before treatment.

Recognition Tools

Use the mnemonic PSL (Prate, Smile, Løfte) — an updated, culturally adapted version of FAST:

  • P – Prate: Dysarthria, language disturbance
  • S – Smile: Facial asymmetry
  • L – Løfte: Arm/leg weakness
TIAs may fully resolve but indicate high stroke risk — they must be evaluated in a stroke center.

NIHSS Collapsible Reference

Level of Consciousness

0 = Alert, 1 = Not alert but arousable, 2 = Requires repeated stimulation, 3 = Unresponsive.

Best Gaze

0 = Normal, 1 = Partial gaze palsy, 2 = Forced deviation.

Visual Fields

0 = Normal, 1 = Partial hemianopia, 2 = Complete hemianopia, 3 = Bilateral hemianopia.

Facial Palsy

0 = Normal, 1 = Minor, 2 = Partial, 3 = Complete paralysis.

Motor Arm (each side)

0 = No drift, 1 = Drift, 2 = Some effort against gravity, 3 = No effort against gravity, 4 = No movement.

Motor Leg (each side)

0 = No drift, 1 = Drift, 2 = Some effort against gravity, 3 = No effort against gravity, 4 = No movement.

Limb Ataxia

0 = Absent, 1 = Present in one limb, 2 = Present in two limbs.

Sensory

0 = Normal, 1 = Mild-to-moderate loss, 2 = Severe-to-total loss.

Language (Aphasia)

0 = Normal, 1 = Mild-to-moderate aphasia, 2 = Severe aphasia, 3 = Mute/global aphasia.

Dysarthria

0 = Normal, 1 = Mild-to-moderate slurring, 2 = Severe.

Extinction & Inattention

0 = None, 1 = Neglect in one modality, 2 = Profound neglect in >1 modality.

NIHSS Scoring (Collapsible)

Open the items you need. Keep it quick—document a total score and key deficits for your prealert.

Tips: Follow the sequence, avoid coaching, and score the first effort. If abnormal from non‑stroke cause (e.g., trauma), score what you see and note the confounder.
1. Level of Consciousness (LOC) — 0–3
  • 0 Alert
  • 1 Not alert; arousable by minor stimulation
  • 2 Requires repeated/painful stimulation
  • 3 Unresponsive
1a. LOC Questions — 0–2
  • 0 Both correct (month + age)
  • 1 One correct
  • 2 Neither correct
1b. LOC Commands — 0–2
  • 0 Performs both (open/close eyes, grip/release)
  • 1 Performs one
  • 2 Performs neither
2. Best Gaze — 0–2
  • 0 Normal
  • 1 Partial gaze palsy
  • 2 Forced deviation
3. Visual Fields — 0–3
  • 0 No loss
  • 1 Partial hemianopia
  • 2 Complete hemianopia
  • 3 Bilateral blindness (incl. cortical)
4. Facial Palsy — 0–3
  • 0 Normal symmetry
  • 1 Minor
  • 2 Partial
  • 3 Complete
5. Motor Arm (each arm) — 0–4
  • 0 No drift (10 s)
  • 1 Drift before 10 s
  • 2 Some effort against gravity
  • 3 No effort against gravity
  • 4 No movement
6. Motor Leg (each leg) — 0–4
  • 0 No drift (5 s)
  • 1 Drift before 5 s
  • 2 Some effort against gravity
  • 3 No effort against gravity
  • 4 No movement
7. Limb Ataxia — 0–2
  • 0 Absent
  • 1 One limb
  • 2 Two limbs

Only score if due to ataxia (not weakness). Use finger‑nose, heel‑shin.

  • 0 Normal
  • 1 Mild‑moderate loss

9. Best Language (Aphasia) — 0–3
  • 0 No aphasia
  • 1 Mild‑moderate
  • 2 Severe

  • 0 Normal

Destination cue: NIHSS high (often ≥10) or VAN+ → consider direct thrombectomy center if regional time cost is < 30–60 min vs nearest hospital.

Brain Physiology Snapshot

The brain is 2% of body weight but consumes 20% of oxygen and 25% of glucose. Cerebral blood flow is tightly autoregulated between MAP 60–160 mmHg. When flow drops below ~20 ml/100g/min, ischemic injury begins — hence the urgency in stroke care.
